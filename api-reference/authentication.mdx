All Videograph APIs are secured with HTTP Basic Authentication. Each request must include your API access token and secret that are generated for your organization’s environment.

## Authentication Method

### HTTP Basic Authentication

Use your Videograph API access token as the username and your secret as the password. Most HTTP clients support Basic Auth natively.

**Header format**:
```
Authorization: Basic BASE64(access_token:secret)
```

<Callout kind="info">
  You do not need to manually build the header when using cURL or standard SDKs. Use their Basic Auth helpers (e.g., cURL `-u user:pass`, Python `HTTPBasicAuth`, Go `req.SetBasicAuth`).
</Callout>

**Example requests**:
<CodeGroup tabs="cURL,JavaScript,Python,Go">
```bash
# List contents with Basic Auth
curl -X GET "https://api.videograph.ai/video/services/api/v1/contents" \
  -u "ACCESS_TOKEN:SECRET" \
  -H "Accept: application/json"
```

```javascript
// Using fetch with Basic Auth
const baseUrl = 'https://api.videograph.ai';
const accessToken = process.env.VG_ACCESS_TOKEN;
const secret = process.env.VG_SECRET;
const basic = btoa(`${accessToken}:${secret}`);

const response = await fetch(`${baseUrl}/video/services/api/v1/contents`, {
  headers: {
    'Authorization': `Basic ${basic}`,
    'Accept': 'application/json'
  }
});

const data = await response.json();
```

```python
# Using Python requests with HTTPBasicAuth
import os
import requests
from requests.auth import HTTPBasicAuth

base_url = 'https://api.videograph.ai'
access_token = os.getenv('VG_ACCESS_TOKEN')
secret = os.getenv('VG_SECRET')

resp = requests.get(
    f"{base_url}/video/services/api/v1/contents",
    headers={'Accept': 'application/json'},
    auth=HTTPBasicAuth(access_token, secret)
)

print(resp.status_code)
print(resp.json())
```

```go
// Using Go http with Basic Auth
package main

import (
    "fmt"
    "io"
    "net/http"
    "os"
)

func main() {
    client := &http.Client{}
    req, _ := http.NewRequest("GET", "https://api.videograph.ai/video/services/api/v1/contents", nil)
    req.Header.Add("Accept", "application/json")

    accessToken := os.Getenv("VG_ACCESS_TOKEN")
    secret := os.Getenv("VG_SECRET")
    req.SetBasicAuth(accessToken, secret)

    resp, err := client.Do(req)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    defer resp.Body.Close()

    body, _ := io.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```
</CodeGroup>

## Getting Your API Credentials

1. Log in to the Videograph dashboard.
2. Navigate to your organization’s environment settings.
3. Create an API access token and secret.
4. Store both values securely (e.g., environment variables, secret manager).

**Environment variables example**:
```bash
# .env (never commit this file)
VG_ACCESS_TOKEN=your_access_token
VG_SECRET=your_secret
VG_BASE_URL=https://api.videograph.ai
```

## Testing Authentication

Quickly verify your credentials with a simple list call:

<CodeGroup tabs="cURL,Python">
```bash
curl -X GET "https://api.videograph.ai/video/services/api/v1/contents" \
  -u "ACCESS_TOKEN:SECRET" \
  -H "Accept: application/json" \
  -w "Status: %{http_code}\n"
```

```python
import requests
from requests.auth import HTTPBasicAuth

resp = requests.get(
  "https://api.videograph.ai/video/services/api/v1/contents",
  headers={"Accept": "application/json"},
  auth=HTTPBasicAuth("ACCESS_TOKEN", "SECRET")
)
print("Status:", resp.status_code)
```
</CodeGroup>

## Error Responses

Common authentication errors and how to address them:

| Status Code | Description | Resolution |
|-------------|-------------|------------|
| `401` | Unauthorized (invalid token/secret) | Verify your access token and secret; regenerate if needed |
| `403` | Forbidden (insufficient privileges) | Ensure the token is scoped to the correct environment/products |

**Example 401 response (illustrative)**:
```json
{
  "status": "Error",
  "code": 401,
  "message": "Unauthorized"
}
```
